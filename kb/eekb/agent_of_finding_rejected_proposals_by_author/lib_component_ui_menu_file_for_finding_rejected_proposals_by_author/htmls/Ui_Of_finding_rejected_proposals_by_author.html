Команда пользовательского интерфейса для поиска отклоненных предложений конкретным разработчиком предназначена для поиска отклоненных предложений. Единственным аргументом команды является разработчик, для которого необходимо найти все отклоненные предложения. Результатом выполнения команды является вывод на экран всех отклоненных предложений конкретного разработчикa.
