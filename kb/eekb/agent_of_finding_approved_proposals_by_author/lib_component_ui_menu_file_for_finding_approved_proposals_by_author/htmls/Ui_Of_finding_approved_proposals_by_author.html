Команда пользовательского интерфейса для поиска утвержденных предложений конкретным разработчиком предназначена для поиска утвержденных предложений. Единственным аргументом команды является разработчик, для которого необходимо найти все утвержденные предложения. Результатом выполнения команды является вывод на экран всех утвержденных предложений конкретного разработчикa.
