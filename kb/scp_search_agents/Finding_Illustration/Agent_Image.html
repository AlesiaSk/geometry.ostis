Задачей агента поиска изображений для заданного понятия является нахождение изображений для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска изображений для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти изображения. Результатом выполнения запроса является множество изображений , найденных для понятия, само понятие и множество связывающих их дуг с отношением трансляция sc-текста* и атрибутом пример' .
