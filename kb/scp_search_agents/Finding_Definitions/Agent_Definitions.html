Задачей агента поиска определения/пояснение для заданного понятия является нахождение определений для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска определения/пояснения для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти определение либо пояснение. Результатом выполнения запроса является определение либо пояснение , найденное для понятия с их основными идентификаторами* , само понятие и множество связывающих их дуг с отношением трансляция sc-текста* и атрибутом пример'.
