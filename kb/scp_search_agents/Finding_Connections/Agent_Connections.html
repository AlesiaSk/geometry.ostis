Задачей агента поиска непосредственных связей между двумя понятиями является нахождение непосредственных связей между двумя понятиями. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска непосредственных связей между двумя понятиями. Аргументами запроса являются два понятия, между которыми необходимо найти все существующие непосредственные связи.
