Задачей агента поиска области определения для заданного отношения является нахождение области определения для заданного отношения. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска области определения для заданного отношения. Аргументом запроса является отношение, для которого необходимо найти область определения. Результатом выполнения запроса является область определения* отношения , найденная для понятия, само отношение и все домены отношения.
