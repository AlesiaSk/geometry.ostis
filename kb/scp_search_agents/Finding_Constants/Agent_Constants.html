Задачей агента поиска констант, через которые определяется заданное понятие является нахождение констант, через которые определяется заданное понятие. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска констант, через которые определяется заданное понятие. Аргументом запроса является понятие, для которого определяются константы. Результатом выполнения запроса является узел, включающий множество констант, через которые определяется заданное понятие, связанный отношением используемые константы* с заданным понятием.
