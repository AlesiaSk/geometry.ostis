Команда поиска констант, через которые определяется заданное понятие предназначена для нахождения констант, через которые определяется заданное понятие. Единственным аргументом команды является понятие, для которого определяются константы. Результатом выполнения команды является узел, включающий множество констант, через которые определяется заданное понятие, связанный отношением используемые константы* с заданным понятием.
