Задачей агента поиска отношений для заданного понятия является нахождение отношений для заданного понятия. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу поиска отношений для заданного понятия. Аргументом запроса является понятие, для которого необходимо найти все отношения. Результатом выполнения запроса являются отношения , найденные для понятия, само понятие и множество связывающих их дуг.
